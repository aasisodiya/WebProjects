Help

Bookmarks is a SPA that helps you to store all Bookmarks in one page so that you can view them all together.

To Add Category : Go to Menu > Click Add Category > Enter Category Name > Click on Add Sign > Done

To Add a Link : Switch On Edit Mode by Clicking Edit Mode Button > Click on Add Sign in required Category > Enter Bookmark Name and URL and click Add Sign

To Edit/Delete Category : Switch On Edit Mode by Clicking Edit Mode Button > Then Click on Pencil Sign > Now in case of edit Enter new Name for Category and click on Edit Category and in case you want to delete the whole category and links inside it then check the given checkbox and click on Delete Category button

To Edit/Delete Link : Switch On Edit Mode by Clicking Edit Mode Button > Then hover on the link that you want to edit/delete then click on delete icon > Then confirm deletion > Done (For edit just delete and add the new link)

To Export/Import Bookmarks : Go to Menu > Click on Export/Import JSON > Copy the data for exporting / Paste your own data for importing and click import
